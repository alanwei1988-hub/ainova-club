# AINova Club 🧪

**青年 AI 创新社群 | Youth AI Innovation Community**

> **Version:** V1.04 (Registration Modal + Form)

> 技术实践 · 项目共创 · 创业连接

---

## 📖 项目介绍

AINova Club 是一个面向高校学生的跨校 AI 创新社群，由启迪之星（上海）发起。

我们致力于：
- 🧠 **技术交流** - 围绕模型、Agent、多模态、具身智能等方向开展持续交流
- 🔨 **项目共创** - 通过项目征集、算力支持、导师交流、场景对接等方式培育创新团队
- 🚀 **创业连接** - 为优秀学生项目提供孵化通道和资源支持

---

## 🎯 核心活动

| 活动 | 描述 |
|------|------|
| **AI Night** | 青年 AI 技术分享夜 - 前沿技术趋势讨论、开源项目分享、产品 demo 展示 |
| **Build Day / Hack Day** | 项目共创日 - 围绕主题组队共创、快速开发和现场展示 |
| **Founder / Researcher Mixer** | 青年创新者交流会 - 创业者、工程师、研究者与学生面对面交流 |
| **Outdoor Meetup** | 轻社交联谊活动 - 增强成员关系与社群黏性 |

## 🛠️ 技术栈

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS
- **Fonts**: JetBrains Mono, Noto Sans SC
- **Animations**: Canvas API, CSS Animations, Intersection Observer

## 🚀 快速开始

### 本地运行

1. 克隆仓库
```bash
git clone git@github.com:alanwei1988-hub/ainova-club.git
cd ainova-club
```

2. 直接在浏览器中打开
```bash
open index.html
# 或使用任意 HTTP 服务器
python -m http.server 8000
```

3. 访问 `http://localhost:8000`
